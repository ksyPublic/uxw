Tooltip

기본

아래 예시는 마크업 구성 시 반드시 포함되어야 하는 속성을 설명하는 구조입니다. 실제 툴팁 구성 시에는 css style 구조에 맞는 마크업이 필요합니다.

* Tooltip은 mouseleave, mouseenter로만 제작되었습니다.
* Tooltip의 data-placement의 옵션으로는 right, left, top, bottom이 있습니다.
* 툴팁생성은 class="tooltip-button", 툴팁[id] => 툴팁버튼 내부의[aria-describedby]와 매칭
