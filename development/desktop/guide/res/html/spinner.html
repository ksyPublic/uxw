Spinner

로딩 스피너

로딩 스피너는 호출 한 만큼 hide가 호출되어야 정상적으로 화면에서 사라집니다.
ex) show 4번 호출 시 hide 역시 4번 호출되어야 합니다. 

                      
                        // 스피너 노출
                          UXW.Spinner.show();

                          setTimeout(function() {
                            // 스피너 삭제
                            UXW.Spinner.hide();

                            // 스피너 강제로 삭제
                            // UXW.Spinner.hide(true);
                        }, 1500);
