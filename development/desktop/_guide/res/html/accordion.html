아코디언 구성 주의사항(웹 접근성 관련)

아래 예시는 마크업 구성 시 반드시 포함되어야 하는 속성을 설명하는 구조입니다. 실제 아코디언 구성 시에는 css style 구조에 맞는 마크업이 필요합니다.

* 아코디언 헤더의 [aria-controls] => 아코디언 컨텐츠의 [id]와 매칭
* 아코디언 헤더의 [id] => 아코디언 컨텐츠의 [aria-labelledby]와 매칭

페이지 로드 시 특정 아코디언을 펼쳐진 상태로 보기

특정 아코디언 콘텐츠를 기본적으로 활성화시키는 방법입니다.

데이터 속성을 이용하여 활성화 시키는 방법

아코디언이 하나만 열려있는 경우는(단수) | data-ui-accordion 이 선언된 엘리먼트에 data-accordion-default="0"

아코디언이 여러개 열려있는 경우는(복수) | data-ui-accordion 이 선언된 엘리먼트에 data-accordion-defaults="0,2"

각 아코디언 헤더에 클래스를 부여하여 활성화 시키는 방법

아코디언 헤드에(aria-expanded 속성이 선언되어 있는 곳) is-active 클래스 추가
